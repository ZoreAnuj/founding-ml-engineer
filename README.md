# Founding ML Engineer Skills

This project explores open-source Claude Code and Cowork skills for sales, recruiting, and growth, leveraging real-time B2B data. It serves as a technical prototype for building intelligent, data-driven workflows in the Crustdata domain.

## Key Features
*   Implements core skills for sales outreach and recruiting automation.
*   Integrates with real-time B2B data APIs for enriched context.
*   Provides a modular framework for developing and testing new AI-powered skills.
*   Includes example prompts and workflows for common growth tasks.

## Tech Stack
*   Claude Code / Anthropic API
*   Python
*   FastAPI (for skill endpoints)
*   Crustdata APIs

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/founding-ml-engineer.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Set your API keys in a `.env` file.
4.  Run the example script: `python examples/sales_workflow.py`
